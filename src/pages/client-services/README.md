Client services workspace

Purpose:
- keeps the client-facing service-request workflow separate from freelancer tools
- creates jobs through the shared `buyer-requests` backend feature
- communicates with proposals and admin through the same platform APIs

Main files:
- `index.jsx`: route entry
- `ClientServicesLayout.jsx`: client shell/navigation
- `ClientOverview.jsx`: system overview
- `CreateJob.jsx`: create service request form
- `MyJobs.jsx`: manage posted requests
